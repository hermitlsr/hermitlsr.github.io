百度收录推送

  本文将介绍如何将自己的博客文章主动推送到百度收录

通过 hexo 框架搭建的个人站，可以被 Google 主动爬取，像我这样的新站，在建站几天之后就被收录了。但是百度站一般不会主动去爬取这类站点，需要站长主动推送站点文章，当然推送了也不一定收录，无奈。

百度收录目前可以通过三个方式：API 提交、sitemap 和 手动提交，本文主要是介绍通过插件实现前两种方式的自动提交。

API 提交

这里主要使用的是 hexo-baidu-url-submit 插件，它可以在每次使用 hexo s 或 hexo d 命令时主动将 _post 内的文章推送到百度
